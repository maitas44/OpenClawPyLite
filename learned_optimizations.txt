- FOCUS VERIFICATION: Always click an input field before typing to ensure focus.
- STATE ANCHORING: Identify a unique text string or icon on the page to confirm a successful navigation or click.
- OVERLAY CLEARANCE: Treat the first action on a new domain as a potential 'Pop-up Clearance' step.
- REDUNDANT NAVIGATION: If a search engine returns zero results, re-verify the search query for typos or pivot to a different engine.
- OFFSET RETRY: If a [x, y] click fails, the subsequent attempt must include a 3-5px coordinate offset.
- LAZY-LOAD TRIGGER: After scrolling down, perform a 'read' action to wait for dynamic content to populate before interacting.
- DOM-VISUAL SYNC: If coordinates appear valid but interaction fails, use 'read' to check if the element is disabled or hidden in the DOM.
